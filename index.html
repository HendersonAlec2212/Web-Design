    Summary: Latitude vs. X

    The purpose of this project was to anaylze the change in weather attributes as one approaches the equator. To accomplish this we first pulled weather measurements from 500 cities.

    After assembling the dataset, we used Matplotlib to plot each of the following: temperature, cloudiness, wind speed, and humidity versus the latitude resulting in a handful of neat visulatizations.

    This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
